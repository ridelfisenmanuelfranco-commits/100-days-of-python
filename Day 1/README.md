# 🪨📄✂️ Día 1 - Piedra, Papel o Tijeras

Bienvenido al primer proyecto de mi reto **100 Days of Python**.

En este ejercicio desarrollé una versión sencilla del clásico juego **Piedra, Papel o Tijeras**, donde el usuario juega contra la computadora.

---

## 🎯 Objetivo

Practicar los fundamentos de Python mediante un programa que permita:

- Recibir datos desde el teclado.
- Generar una opción aleatoria para la computadora.
- Comparar diferentes condiciones.
- Mostrar el resultado de la partida.

---

## 📚 Temas Practicados

- Variables
- Listas
- Entrada de datos (`input`)
- Salida por consola (`print`)
- Condicionales (`if`, `elif`, `else`)
- Operadores lógicos (`and`, `or`)
- Módulo `random`
- Métodos de cadenas (`lower()`)

---

## 🚀 Cómo ejecutar

1. Clona el repositorio.

```bash
git clone https://github.com/TU_USUARIO/100-days-of-python.git
```

2. Entra a la carpeta del proyecto.

```bash
cd day-001
```

3. Ejecuta el programa.

```bash
python main.py
```
